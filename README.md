# @64labs/sassy

> A collection of React components and styles

## Install

```bash
npm install --save @64labs/sassy
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from '@64labs/sassy'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
