## Usage & customization

The Icon component's `name` prop references symbol IDs in an SVG sprite. Here we're using the full sprite sheet from [Feather Icons](https://feathericons.com/). You can use these icons in your app by including the provided SVG sprite sheet or customize them by including your own. You'll want to ensure that the symbol IDs you use match ones used internally (for instance, `Checkbox` references an SVG symbol ID called `check`).
